# Data Directory

This folder intentionally contains no EEG data.

Download ERP CORE P3 dataset from:
- **OSF**: https://osf.io/etdkz/ (Original format with analysis scripts)
- **OpenNeuro BIDS**: https://openneuro.org/datasets/ds003061/ (BIDS-compatible format)

Place raw files in: `data/raw_p3b/`
Processed files will be created in: `data/processed_p3b/`
